# Changelog

## 1.0.0 — 2024-03-19

Initial open-source release.

### Features
- Random sound playback by category (success, fail, neutral, misc)
- Event listener with logging and macOS notifications
- Git hooks integration (post-commit, pre-push, post-merge, post-checkout)
- Global git hooks setup for new repositories
- Claude CLI integration for AI-assisted workflows
- Test runner wrapper (`test-with-sound`)
- Bumblebee-style voice clips with radio effects (via `sox`)
- SoundBible robot/mechanical sound downloader
- Freesound.org API integration for custom sound discovery
- YouTube clip extraction tool (for personal use)
- macOS system sound integration
